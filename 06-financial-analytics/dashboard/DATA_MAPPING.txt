============================================
FORMATO DE DADOS ESPERADO
============================================

CURRENCY: Valores monetários em USD (ex: 1000000)
DECIMAL: Números decimais (ex: 1.75, 3.45)
PERCENT: Percentuais já em decimal (ex: 15.00 = 15%, 3.5 = 3.5%)
DATE: Formato ISO YYYY-MM-DD (ex: 2025-09-30)
TEXT: Texto livre (ex: "North", "Product A")
INTEGER: Números inteiros (ex: 15000)

============================================
DICAS DE OTIMIZAÇÃO NO POWER BI
============================================

1. Use Power Query para limpeza adicional se necessário
2. Crie uma tabela de Calendário (dimensão de tempo)
3. Adicione colunas calculadas para análises adicionais
4. Use medidas DAX para KPIs dinâmicos
5. Implemente filtros globais em cada página
6. Padronize cores e formatação entre visualizações
7. Use tooltips para informações adicionais
8. Crie bookmarks para diferentes cenários de análise
